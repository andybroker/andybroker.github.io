一项新闻报道显示分红并回购股票的公司具有股价优势


轨道科学公司Cignus号航天飞机正向国际空间站靠拢

既支付股息又进行股票回购对公司股价具有很大提振作用，位于纽约杰富瑞投资银行分析师这样说道，较少的股票数量意味着每股净利润的增加。

银行利息率处于长期低位，全世界数十亿美元政府基金都是负收益率，相比之下股息收益具有吸引力，今年标准普尔500指数股票表现最好的是通信领域14.9%的收益率和公用事业板块13%的收率

分红及股票回购案例分析

全世界资金泛滥，2008年信用危机后联邦储备银行及其他央行进行一轮又一轮刺激，美国较低的借款成本及长期经济复苏意味着国内中小企业手握大量现金，“身材妖娆”，杰富瑞投资银行股权策略师Steven DeSanctis这样说道。

周一一份报道里DeSanctis说收购案例的增加减少了美股数量，自1984年以来已达到最低水平。

“去年我们发现有31%的小盘股公司减少了流通股份，从2011年大盘股公司就已经减少流通股份。”

通过他的银行数据以及FactSet和Rusell投资集团提供的信息，DeSanctis和他的研究团队计算出从1985年底，现金分红的同时还进行股票回购的公司股价表现显著优于美股整体表现：

1-进行股票回购的公司获得了年华14.1%的收益，而增发股票或者维持原来股份数量的公司年华收益为6.7%
2-进行现金分红的公司年华收益为11.7%，未分红公司收益为6%
3-既分红又回购股票的公司年华收益为13%，其他公司整体收益为9%。
当公司过度分红或者董事会以股票回购代替拓展公司规模时的时候股价并不总是有好的表现。

美股中的现金帐户与融资帐户区别

发表于 2016-07-19   |   分类于 美股开户   |     |   阅读次数

现金帐户
投资者可以申请及透过现金帐户交易股票，ETFs，共同基金等等。但是, 在现金帐户中交易的股票资金需要三个交易日来完成交割。交割一词代表证券被正式转移至买方的帐户，且资金正式进入卖方的帐户。期权交易只需要一个交易日来完成。当使用现金帐户，投资者需要按照现金帐户规则来完成交易。如果使用未完全交割资金，投资者有可能会收到Good Faith Violation违规操作的警告和90天的帐户限制。使用融资帐户可以帮助您避免类似的警告和限制。
