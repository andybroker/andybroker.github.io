如何培养孩子的阅读兴趣？

发表于 2017-04-09 | 分类于 Quora精选 | | 阅读次数

翻译：Andybroker


8岁的时候我根本不喜欢读书。父亲是一位极富洞察力的老师（白天是大学老师，晚上是8岁孩子的幼儿教师）他是这样解决这个问题的：

我当时甚至没意识到他正设法让我爱上阅读，数年后我把各种举动罗列在一起才察觉到。

每周，我的房间里都会出现一本从图书馆借来的书，各种各样的类型，有的是专门写给孩子看的，有些则不是。我无法再详细描述这些书籍了，因为我压根儿没看过他们，当时并没有觉得有压力，我只是习惯了他们会出现在我房间（小孩子很容易接受他们周围的环境）。

父亲一直这样做，重复着，周复一周，月复一月，他只说过一句：“有空就看看。”

直到后来他得逞了。《纳尼亚传奇：狮子、女巫与魔衣橱》，作者：C·S·路易斯，我神情激动地跑下楼梯，嘴里念叨着一个衣橱可以通向冰雪覆盖的世界。

父亲并没有夸赞我，他听到这个消息的时候假装若无其事，起身离开时说了句：“当你读到狮王阿斯兰时，要知道C·S·路易斯是一位虔诚的基督徒”（阿斯兰集基督教中的圣父，圣灵，圣子三位一体，在七部曲中的每一部里都有出现，是贯穿首尾的角色，具有崇高的地位，是纳尼亚王国的拯救者，海外皇帝之子，而不仅是人们认为的普通狮子。应该说，阿斯兰是纳尼亚故事的真正主角。《魔法师的外甥》中，他创立了纳尼亚王国并且教迪格雷（也就是后来的柯克教授）种下了纳尼亚著名的果树；《狮子女巫和魔衣橱》中他回到纳尼亚，拯救了冰雪笼罩下的纳尼亚，为四位国王和女王加冕；《能言马和男孩》中，他帮助纳尼亚南部国家，阿钦兰的王子回到自己的国家；《凯斯宾王子》中，他助年轻的王子重登王位，让古老的纳尼亚复活；在《黎明踏浪号》中，他指引黎明踏浪号走上正确的路；《银椅》中，他帮助尤斯塔斯和吉尔找到凯斯宾的独子瑞廉；,《最后一战》中，他让纳尼亚毁灭并且带领纳尼亚的朋友们迎接“真实的纳尼亚”……应该说，缺少阿斯兰，纳尼亚故事就失去了其核心所在。译者注）

我当然读完了它，随即它就消失了，再没有新的书出现。我问父亲：“这周没有书吗？”
“看下你的衣橱，那里是通向神奇王国的大门。”

纳尼亚整部系列都堆放在那里，没有图书馆的印章，这是父亲花钱买来的。

几个星期时间我读完了整部系列。

接下来出现在我房间里的是《时间的皱纹》，父亲猜我喜欢奇幻作品。

当他我为找书的时候，碰巧发现了我喜欢科幻作品（雷·布莱伯利、亚瑟·克拉克），并留意到我尤其喜欢英国作家。

父亲并不担心8岁大的孩子的接受能力。我阅读世界末日系列书籍，像《魔王之槌》、反乌托邦科幻小说《美丽新世界》。

就这样，阅读伴随我度过了9岁、10岁，在书店或图书馆，手里捧着一摞书任凭自己放松。
