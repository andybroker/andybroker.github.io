如果希特勒如此罪大恶极，为什么他周围的人不直接杀了他？

发表于 2017-02-08 | 分类于 Quora精选 | |

翻译：Andybroker

所需要的仅仅是一颗可以射穿他脖子的子弹，当然，其他警卫会立刻杀了你，但你难道不愿意杀掉希特勒吗？虽然会有牺牲，却可以成为一名英雄。

这个最残暴的独裁者到底从哪里获得的权力？是什么让如此多的人成为他的追随者？

因为希特勒对他们并不坏

暂时以普通德国人的角度换位思考一下

你的国家输掉了第一次世界大战，任由胜利者摆布，没有获得丝毫怜悯。现在不得不支付赔款，军队也受到了极大限制。[1]

更糟糕的是，他们把一切归咎于你，德国被强制承担所有的战争责任，甚至被剥夺了辩解的权力。[2]

你的国家自豪感破碎了，在国际舞台上感受到了屈辱。

随之而来的是经济大萧条，生活如此糟糕，你无法养活你的家人，周围的人境遇都很悲掺，政府努力恢复经济，却一次又一次失败，生活变得越来越糟。[3]

这时，一个家伙登场了

他承诺要恢复德国军队以前的荣耀，他那样做了。[4]

他承诺要收复战争中失去的领地，他那样做了。[5]

他承诺要废止《凡尔赛条约》的耻辱，他那样做了。[6]

他承诺要创造更多就业，帮你把食物端上餐桌，他那样做了。[7]

最重要的是，他给了你民族认同感。你又可以以国家为荣了。

所以你拥戴他是很自然的事。更别提你参加了他的大规模集会，被他的权力、富有激情的演说冲昏了头脑。

每个你认识的人——朋友、邻居、老板都拥戴他。

他让德国再一次变得伟大。你怎么会去刺杀这样一个人呢！
