你做过的最艰难的事是什么？

发表于 2017-02-07   |   分类于 Quora精选   |     |   阅读次数

翻译：Andybroker

不再逃避

这个问题下面已经有一些很棒的答案了，但我将说下我自己的经历。因个人原因，决定匿名回答。

小时候，我的生活相当不错，处于中上阶层，双亲家庭，雇得起保姆，有许多宠物，该有的都有了。问题是这一切都是幻像，所有的钱都是借来的，所有的奢华都是我父母用来掩饰正在崩溃的生活的门面。

六岁的时候母亲遭遇了车祸，造成神经损伤，得服用很大剂量的止痛药——稍后我们再说这个话题——最直接的影响就是这次事故触发了她童年被压抑的记忆，接下来的一两年（具体多久记得不太清楚了）常常出入精神病院。

我清楚的记得那年我正躺在床上，门裂了道缝，母亲趴在走廊里，手脚并用，拖着父亲的保险箱，嘴里不停的大喊“他不会要我们了”，“他不会要我们了”。这样的情况持续了一段时间。当时我假装自己不在那里。

七岁的时候，父亲也遭遇了车祸，不过并未受伤，可他以此为借口开始看精神科医生，抱怨自己头疼，显然他总是头疼。他也拿到大剂量的止痛药，几个月内他们两人服用的氢吗啡足以杀死一匹马。即便七岁的年纪，我也能轻易察觉到他们失去了往日的光辉。

我有个比我年纪稍大的姐姐，一年后她证实了我的猜测——父母二人得了长期药物成瘾，只能短暂性戒除。我不知道该怎么办，所以假装自己不在那里。

八岁的时候，父母送我去了一私立学校。他们不仅对药物成瘾，而且还迷恋财富和社会地位，送我去私立学校便是出于这样的虚荣心，我并不适应那里的生活，成绩一落千丈。

第一份成绩单出来后，父亲抽出皮带就打，我以前从没被打过。我记不得被皮带抽打的时刻，不记得那些掌掴，也记不得坚硬的拳头，只记得身体像秋千一样晃动，只记得脸颊贴在地毯上，被泪水沾湿，唯独不记得那些挨过的打。

每次被父亲打的时候，母亲的反应都是一样的，流泪、身体抽搐，在几米开外冷眼观看。有时候她会在结束挨打后走过来跟我说“是你毁了我”，我很久以后才弄明白那是什么意思。

我猜想因为我假装自己不在现场，是否就永远不记得痛苦的时刻。

十一岁的时候，我父亲工作上遭受了重大挫折，我们搬去了新的城镇，换了便宜点的房子，不再有那么多车，奢靡之费正越来越少，门面开始坍塌，肮脏的内里开始显露。那个时后我父母的大部分时间都在嗨，我猜部分是因为要面对生活不幸的压力，部分是因为本能，那是一个恶性循环，他们逐渐精神失常。

父亲最先失控。一天我被巨大的响声和叫喊声吵醒，父母吵得很厉害，我像往常一样假装没有在那里，尽管这次有些不同寻常，我能察觉。我走出房间，之后的事记得有点模糊，不知道是否因为情绪亢奋而短暂失忆，还是把记忆埋藏了起来。

离开房间后的事情我倒记得，母亲跟我跑下楼梯，手里都拿着好几把枪，父亲在后面追赶，光着身子，一直吼，他想要杀了我们。他想拿枪，但我们抢先拿走。一生中我从未如此害怕。

他热衷收集枪械，尽管有滥用药物的历史，且精神不稳定。拜他所赐，我成了控枪法案的坚定支持者。

有时，我想起印第安纳琼斯电影里的滚落的巨石场景，真的很像，只不过我们手里拿的不是宝藏，而是枪。追赶我们的也不是滚落的巨石，而是想杀了我和母亲的疯子。

我们往外跑，弄翻身后的物品，无论如何也不能把枪落下，后来成功躲进了母亲的车里，把枪锁在了里面，我俩知道这并不能阻止父亲，但可以此拖住他，那正是我们想要的，我们钻进车里后不久警察就来了，父亲还在挣扎，可是当时来了至少六辆警车，他没有机会了，也就是那天，我亲眼目睹父亲被警察带走。

我记不得接下来的几天发生了什么，我努力假装自己不在那里。

十四岁时，我去探访父亲，他们总是监督访客，所以母亲也得跟着。出来的时候，母亲让我在车里等着，他俩在入口通道待了几分钟，我听到了争吵声，紧接着母亲要离开，我看见父亲抓着她的胳膊，拧到后背上，抓着她的头撞击车库的门。

我不知道该干什么，所以我做了我想做的

我下车跑向他们两个，冲着父亲吼，这是我第一次吼人，而不是大喊，那是很恐怖的声音，现在每次要吼人都会想想那个时候，与之相比吼得还不够。

父亲对吼声无动于衷，于是按照逻辑采取了行动

我打了他

一下

一下

又一下

我用胳膊锁住他的脖子，把话又讲了一遍，很快他停手了，没有再伤害我们，我跟着母亲离开了那里。

那天，我没有假装自己不在那里。

十五岁时，母亲开始酗酒，父亲被带走后她已经清醒好几年，但又开始变得糟糕，而且很快。她工作有困难，一直酒驾，最终闯了车祸，断了几根骨头，日渐消瘦，她正在杀死自己。

我们没有钱，没食物，她得到工作只能维持工作一到两周，拿到的钱都用来购买廉价的酒精，我去街上乞讨些零用钱，攒够了就去加油站买些煤油用来取暖，天开始下雪以后生活就更困难了，我们没有暖气。我尽可能待在朋友家里，他们的父母会让我跟他们一起用餐，没有人问我什么，也不必问。那年一月份我偷钱买食物，多余的钱就买些药，但那根本无法维持生存。

二月份，母亲比之前喝得更醉了，她一直告诉我真希望我没有出生，我是个错误。她告诉为什么决定要孩子，父亲那时想要离婚，所以她以孩子为要挟，我是她的工具。现在我才知道为什么她总是说是我毁了她的一切。

后来，语言已经不够，她拿起小刀戳我脖子，我很震惊，差点没有及时躲过去。幸亏她喝醉了，我也已经15岁，行动敏捷，我抓住她的手腕，从手中夺过了刀子，我把自己锁在浴室里，直到想清楚该干什么。

我想要假装自己不在那里，但是我不能

相反，我叫了警察，人生中第二次看着警察带走了父母。

孤身一人了，不知道该干什么

我打电话给了父亲

十六岁时，我跟父亲住在了一起，他已经清醒了，至少比以前好了许多。我们跟他的女朋友生活在一所富人区的大房子里。他恢复了往日的魅力，我的房间——确切的说是楼梯下面的一个大的衣柜，那年我读了很多遍《哈利波特》，让我感觉好了点。

那时候我跟父亲之间还算不错，至少我想让一切和缓些。他是一个病态撒谎者，利用一切机会控制他人，这一点没变，但这比寄养好多了，也比无家可归好多了。那时很难交到朋友，但我在等待时机来临，我知道会成功。

然后我被强奸了。

那时，我不认为男人可以被强暴，至少不会被女人强暴。我花了好长时间才承认这是事实。实际上，知道20岁我才能真正启齿——我被强暴了。一次又一次，之所以很难称之为强奸：一是因为我坚持秉承那个年代的性别角色观念；二是这发生在亲密关系之间，是虐待性关系，我并不知晓，不知道这是错的。

我以为我是错的

只要一发生，我就闭上眼睛，假装自己不在那里

然后，有一天，我发怒了

二十岁时，我厌倦了逃避，厌倦了演得像自己没有受伤一样，厌倦了让人们摆弄我，所以我停止了一切。

一个接着一个，结束了所有的虐待性关系

我打电话给我的伴侣，告诉她再也不想见到她了，那个时候我还没有称之为强奸，但我清楚那是错误的，我知道我得结束这一切。

我打电话给我母亲，告诉她我再也不想见到她，她尝试杀死我以后我曾试着修复我们之间的关系，但那永远都不会见效，她永远都看不到她做错了什么，总以为是我影响了她的人生，那不是我的错。

我打电话给我的父亲，告诉他我再也不想见到他，十八岁前我得忍受跟他生活两年，但仅仅是因为我一直假装自己不在那里，离开他很难，因为我不认为他是个坏人，只是极度自恋，不能自拔，对他周围的人难以置信的危险。那两年里他又差点杀了我，但我让他冷静了下来。我清楚，无论他的动机是什么，跟他在一起我别想安全。

除此之外，我确实很想念父亲，确切的说，我很想念当初的那个他。

正是那时我不再逃避

再也不假装自己不在现场，因为我的确在那里

离开是最好的选择，也是我一生中做过的最难的选择，远超任何一场噩梦

但我很开心那么做了

二十一岁时，我开始了毕业班的第一学期，五年半后，我过得的确很好，我进行心理治疗，已经有几年了，很有帮助，但还有许多阴影需要摆脱，我有很好的朋友，健康的关系，完美的生活。还有三个星期就要博士论文答辩，然后开始激动许久的工作，我成功让生活走上正轨，深深地感恩。

告别是如此难以置信，相当困难，但这对我很有意义

我再也不逃避了

现在我就在这里。
