你最后悔的事是什么？——来自Quora匿名用户的回答

发表于 2018-03-24 | 分类于 Quora精选 | | 阅读次数

翻译：Andybroker


最后悔大学最后一年在Facebook上给我迷恋的人说了“嗨！”

她是个有智慧、有气质的女孩儿，整个大学期间我对她都很敬畏，从来没有勇气跟她开始一场对话。

不知怎的，她拿到就业录取通知那天我鼓起勇气跟她打招呼，并表示祝贺。

故事就这样开始了，她回复了我，然后我们开始了谈话。在Facebook上我们每隔几天就会聊聊，话题都很随意，从那天起对话就没停止过。有几次我聊着聊着就睡在了自己笔记本电脑上，我们从Facebook慢慢转移到whatsapp上，并渐渐成为了朋友。六个月的深入相处，友谊开始显现出占有欲的迹象。我们都清楚彼此已经不仅仅是朋友了，但很长一段时间我们都不能接受对方，因为我俩家东正教的家庭背景不会理解我们的爱。

两年后在一次偶然的争吵中她承认爱上了我，我们不再单单只是朋友了，这几年我都爱着她，是我们之间的宗教背景阻止了我进一步发展，但是，现在她传达了自己的真情实感，我也就什么都不管了，带着对彼此未来的爱与深深的期望，开始了我们的正式关系。

这段关系进行的并不顺利，诸多坎坷，但我们就这样在一起度过了两年半时间，我开始把她当我的妻子而不是女朋友，爱情让我们充满希望，也让我俩盲目。我甚至在这几年时间里跟她计划好了整个一生。

我差不多说服了我的家人，他们渐渐接受了我俩之间的爱情。但她没能说服她的家人。她的父母对她进行情感折磨，用亲情关系绑架，那段时间她经受了很多创伤，那是我们相处的岁月里最具灾难性的三个月。她尽了最大努力，不得不在我跟她父母之间做出选择。我很爱她，可以为她做任何事情，做任何可以令她感觉好一点的事。在她经历了那些情感折磨后我所想的只是让她开心起来，我知道如果她的父母不开心，我们也不会过得幸福，因此我们结束了这段关系。

我根本不清楚这会在接下来几个月给我造成什么影响，因为在那一刻我所想的就只是让她摆脱来自父母强加给她的压力。分手最初几天还好，我消磨大量时间在工作上，或是跟朋友在一起，可是渐渐的分手的事实逐渐发挥了它的效力，三个月后我想要去看看我曾经跟她在一起的地方，从那时我一一回去那些地方，开始想她想得厉害，跟人们之间的谈话变少了，每晚花很长时间独自盯着空洞的天空。

分手六个月后，我得知她已经结婚了，为她和她的父母感到开心，同时也认知到分手与看着她牵手别人完全是两码事。我内心变得麻木，本属于你的东西被突然从身上撕掉，很难过，体重下降，对生活失去憧憬，连续几个月没有发自内心的微笑。我现在也在努力的应对人生，为了我的父母我要撑下去。

差不多分手已经一年了，现在是凌晨三点，反思五年前如果我没有在Facebook上发送那个“嗨！”一切会是怎样的？或许我不用承受着来自她现在和谐的家庭带给我的痛苦与感情负担；或许我现在不用思念着别人的妻子；或许我现在可以心无杂念的安详睡觉。是的，真后悔那天跟她推送消息。

全文完。
